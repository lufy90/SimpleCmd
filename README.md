# command line agent

## Start

1. install requirements

```bash
pip install -r requirements.txt
```

2. set API KEY in `.env`

```bash
AI_API_KEY=your_api_key_here
AI_API_URL=https://api.openai.com/v1/chat/completions
AI_MODEL=gpt-3.5-turbo
```

3. run

```bash
python main.py
```

## Build in commands

- `help`
- `history`
- `clear`
- `exit` / `quit`
